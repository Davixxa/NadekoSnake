# NadekoSnake

BASH inside "react-frontend" and "react-backend"

 $ npm install

to run the apps
 $ npm start
however it is recommended to use nodemon

 $ npm install -g nodemon
 
and do

 $ npm nodemon
 
to run the api with nodemon, do not do it with the frontend.

inside the frontend folder
